файлы проекта:
* model.ipynb блокнот с кодом обучения модели
* train.csv обучающий набор данных
* pipelines.pkl сериализованные конвейеры моделей
* pipeline.py класс конвейера модели
* targetpay.py REST API Web - сервис
* Dockerfile - конфигурация образа docker
