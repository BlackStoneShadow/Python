# Инструкция
## Описание
Программа предназначена для ведения телефонного справочника
## Меню
* A_ll выводит весь список телефонов
* N_ew добавляет новый телефон в справочник
* R_emove удаляет телефон из справочника
* F_ind поиск телефона по номеру в справочнике
* S_ave сохранение справочника в файл
* E_xit выход из программы

выбор требуемого действия осуществляется вводом первой буквы названия пункта меню
## Формат хранения данных
Программа поддерживает два типа формата хранения данных :
* json
* csv

Формат хранения определяется в момент инициализации модуля модели:

        Model.PhoneBook('PhoneBook.json')
        Model.PhoneBook('PhoneBook.csv')

## Архитектура программы
Программа разработана по архитектуре MVC, в ее состав входят три модуля:
* модель(Model.py)
* контроллер(Controller.py)
* представление(View.py)

Взаимодействие между ними показано на схеме
