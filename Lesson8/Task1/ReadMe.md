# Инструкция
## Описание
Программа предназначена для ведения списка сотрудников 
## Меню
* A_ll выводит полный список сотрудников
* N_ew прием нового сотрудника
* R_emove увольнение сотрудника
* T_ransfer перевод сотрудника в другое подразделение
* F_ind поиск сотрудника по социальному номеру
* S_ave сохранение списка сотрудников в файл
* E_xit выход из программы

выбор требуемого действия осуществляется вводом первой буквы названия пункта меню
## Формат хранения данных
Программа поддерживает хранение данных в формате json
## Архитектура программы
Программа разработана по архитектуре MVC, в ее состав входят три модуля:
* модель(Model.py)
* контроллер(Controller.py)
* представление(View.py)

Взаимодействие между модулями показано на схеме    

Программа поддерживает дтнамически расширяемый набор атрибутов сущности сотрудник для добавления которых достаточно перечислить их в результате возвращаемом функцией record

        def record(self):    
        return (self.snils(), self.name(), ...)
   каждый элемент представляет собой картеж с парой элементов:
* наименование поле
* значение поле

Ключевым атрибутом сущности сотрудник является snils, по нему контролируется уникальность записей в списке и осуществляется их поиск.
