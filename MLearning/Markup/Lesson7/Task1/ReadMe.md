URL сайта: https://www.wildberries.ru

Описание: Для этого проекта был выбран сайт крупного интернет-магазина wildberries.ru, информацию с этого сайта позволяет получить достоверную статистику о ценах различных товаров. Данное приложение позволяет получить цены по заданному товару и сохранить их в csv файл для дальнейщего анализа.

Подход: Для навигации по сайту и извлечения данных используется библиотеку Selenium в связке с BeautifulSoup, Selenium выполняет загрузку страниц и пагинацию по ним, а BeautifulSoup извлечение данных. 

Запуск сценария.

    python task1.py -s "IPhone 15 Pro MAX" -f result.csv

Трудности: При работе Selenium с сайтом возникли раяд проблем, поскольку загрузка страниц занимает какое-то время пришлось использовать методы библиотеки ui ожидающие появления элементов страницы в процессе ее загрузки, так же сайт имеет защиту от скрейпинга, которая блокирует доступ с его содержимому в режиме запуска Selenium --headless, что требует отсутствие посторонней активности на компьютере во время исполнения скрипта.

Результаты: в процессе исполнения скрипта формируется файл с актуальными ссылками и ценами на предложения по заданному товару.
