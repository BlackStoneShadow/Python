## Загрузка котировок
Модуль предназначен для получения котировок валют с сайта Центробанка России за период дат. 
Результатом работы модуля является файл csv с полученными котировками за указанный период. 
Формат запуска Task1.py -b "Дата начала" -e "Дата окнчания" -f "имя файла", 
по умолчанию котировки выгружаются за текущий месяц в файл data.csv
